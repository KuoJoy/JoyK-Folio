常見問題

Q : 請問在哪裡上課？上課時間？

訂閱後，請至訂購人信箱收取「訂購成功通知信」，內有課程連結，點擊連結，登入KNACK影音學習平台，進入「從運動學管理：商業實戰MBA！」即可上課。

Q : 「從運動學管理：商業實戰MBA」課程如何運用影音教學？

此課程為雙人互動對談影音，於實體講座空間錄製完成，並非單一講師的棚內錄影，因此課程內並無影音動畫，場景全面為實體講座空間的實錄畫面。

Q : 請問如何加入LINE@課程群組？

訂閱課程後，請至訂購人信箱收取「訂購成功通知信」，內有「今日職場修煉」LINE@群組連結，點擊連結即可加入社團。

Q : 請問訂閱課程後，要如何取得筆記本？

訂閱課程後，每張訂單編號限領乙本，SmartM將於8/11（日）後，以掛號方式陸續寄出，若您未在8/31（六）前收到掛號，歡迎於上班時間與我們聯繫。

Q : 請問我可以挑選筆記本顏色嗎？收到後還可以更換嗎？

本課程贈送之筆記本為無償取得之贈品，恕不得要求指定樣式與更換。
